# Применение условных вероятностей в анализе данных

## Условие задачи
Для анализа выбран набор данных online_shoppers_intention.csv (в репозитории представлен как untitled.csv), доступный на платформе Kaggle:
www.kaggle.com

## Требуется
На некотором наборе данных продемонстрировать применение условных вероятностей
Назначение данных:
Набор данных используется для прогнозирования намерения посетителя интернет-магазина совершить покупку (Revenue) на основе его онлайн-поведения в течение сессии.
