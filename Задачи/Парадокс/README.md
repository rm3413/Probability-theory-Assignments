# Парадокс мальчика и девочки

## Условие задачи
Рассматривается семья с двумя детьми. Известно, что в семье есть мальчик, но информация об этом получена двумя разными способами:

**Сценарий 1:** Известно только, что хотя бы один из детей — мальчик.  
**Сценарий 2:** Известно конкретно, что старший ребенок — мальчик.

## Требуется
1. Вычислить вероятность того, что оба ребенка мальчики для каждого сценария
2. Смоделировать оба сценария
3. Показать разницу в вероятностях
4. Объяснить причину различия

## Математическая основа
Пространство элементарных исходов для семьи с двумя детьми состоит из 4 равновероятных комбинаций:
1. Старший мальчик, младший мальчик
2. Старший мальчик, младший девочка  
3. Старший девочка, младший мальчик
4. Старший девочка, младший девочка

## Ключевой результат
- В сценарии 1 вероятность обоих мальчиков составляет **1/3 ≈ 33.3%**
- В сценарии 2 вероятность обоих мальчиков составляет **1/2 = 50.0%**

Парадокс заключается в том, что интуитивно оба сценария кажутся одинаковыми (в обоих известно о наличии мальчика), но математические расчеты дают разные результаты.
