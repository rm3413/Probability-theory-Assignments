# Связь частотного и классического определения вероятности

## Условие задачи
Рассматривается бросание двух игральных кубиков. Исследуется событие: **сумма выпавших очков равна 7**.

## Требуется
1. Вычислить классическую вероятность события
2. Смоделировать эксперимент
3. Построить график сходимости частотной вероятности к классической при увеличении числа испытаний
4. Показать визуально связь двух определений вероятности
