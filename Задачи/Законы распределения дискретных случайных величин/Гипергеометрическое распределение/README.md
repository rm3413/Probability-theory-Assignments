# Гипергеометрическое распределение

## Условие задачи
В партии 30 плат, из них 8 бракованных. Наугад выбирают 4 платы без возвращения.

## Требуется
а) Составить закон распределения дискретной случайной величины X - числа бракованных плат среди выбранных
б) Найти математическое ожидание, дисперсию, среднеквадратичное отклонение, моду
в) Определить вероятность отсутствия бракованных плат среди выбранных
г) Построить многоугольник распределения
