# Геометрическое распределение

## Условие задачи
Вероятность обнаружения дефекта при проверке изделия равна p = 0,2. Проверка производится до первого обнаружения дефекта.

## Требуется
а) Составить закон распределения дискретной случайной величины (вероятности для k = 1, 2, 3, ...)
б) Найти математическое ожидание, дисперсию, среднеквадратичное отклонение, моду
в) Определить вероятность того, что потребуется проверить не менее 12 изделий
г) Построить многоугольник распределения
