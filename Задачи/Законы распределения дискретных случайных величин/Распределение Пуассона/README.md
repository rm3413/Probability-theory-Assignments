# Распределение Пуассона

## Условие задачи
Число аварий на участке дороги за 1 день — это случайная величина X, имеющая распределение Пуассона с параметром λ = 1,7.

## Требуется
а) Составить закон распределения дискретной случайной величины X при значениях X от 1 до 5
б) Найти математическое ожидание, дисперсию, среднеквадратичное отклонение, моду
в) Вычислить вероятность того, что за 1 день произойдёт более одной аварии
г) Построить многоугольник распределения
