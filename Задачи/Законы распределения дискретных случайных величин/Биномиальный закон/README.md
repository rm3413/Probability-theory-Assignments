# Биномиальное распределение: Перегоревшие фонари

## Условие задачи
Вероятность того, что фонарь перегорит в течение гарантийного срока, равна 0.3. Проверяется партия из 10 фонарей.

## Требуется
1. Найти закон распределения числа перегоревших фонарей (вероятности для k = 0, 1, 2, ..., 10)
2. Вычислить математическое ожидание
3. Вычислить дисперсию
4. Найти моду (наиболее вероятное число перегоревших фонарей)
5. Вычислить среднеквадратическое отклонение
6. Построить многоугольник распределения
