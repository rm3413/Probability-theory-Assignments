# Геометрическая вероятность

**Условие**  
Игрок набрасывает кольца на колышек с закрытыми глазами.  
Кольцо случайным образом падает на квадратный участок со стороной \( a \).  
В центре участка находится мишень — круглый колышек радиусом \( r \).  

Известно, что площадь колышка составляет \( 2/10 \) (или \( 1/5 \)) площади квадрата.  

**Требуется**  
а) вычислить теоретическую вероятность попадания;  
б) смоделировать эксперимент;  
в) построить графическую иллюстрацию;
